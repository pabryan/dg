Write $\gamma(s) = (x(s), y(s))$

<p class="fragment">
Scaling: $\tfrac{2\pi}{L} \gamma$ has length $2\pi$ and the same $L^2/A$.
</p>

<p class="fragment">
Translate: let $x_0 = \int x ds$ and $y_0 = \int y ds$ replace $\gamma$ by

$$
\gamma - (x_0, y_0)
$$
</p>

<p class="fragment">
Then

$$
\int x ds = \int y ds = 0
$$
</p>
