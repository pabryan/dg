The function

$$
n(u) = \frac{1}{2\pi} \int_0^L \kappa_u(s) ds
$$

is continuous and integer valued, hence constant.

<p class="fragment">
$n(1)$ is $\pm 1$ since $\gamma_1(s)$ loops around the unit circle once.
</p>

<p class="fragment">
$\gamma_0(s) = \gamma(s)$
</p>

<p class="fragment">
Thus $n = n(0) = n(1) = \pm 1$.
</p>
